# CRD
The CRD dataset consists of various parameters collected from a steel company in China, representing real-world industrial scenarios. It includes nine types of  parameters includees:rolling forces, bending force, rolling speed, tension, Entry set thickness, Thick_EX, width, Entry actual thickness and Exit actual thickness.The dataset contains ten rolls of steel strip with different widths and thicknesses, totaling approximately 15,000 samples.The dataset details are as follows:
